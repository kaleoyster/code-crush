Login

Requirements

The Login App should function as follows:

  • Users should be able to enter their username and password.
  • Clicking the "Login" button should check if the credentials are correct.
  • If the username is "admin" and the password is "password", the app should display a "Login Successful!" message.
  • If incorrect credentials are entered, an error message should appear.
  • The app should clear error messages when correct login details are entered.
  • The password should be hidden using a password field (●●● instead of visible text).
  • If a user tries to log in without entering both fields, a warning should appear.
